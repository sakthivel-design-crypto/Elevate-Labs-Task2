# Elevate Labs - AI & ML Internship

## Task 2: Exploratory Data Analysis (EDA)

### Objective

The objective of this task is to explore and understand the Titanic dataset using descriptive statistics and data visualizations.

---

## Tools Used

- Python
- Visual Studio Code
- Pandas
- Matplotlib
- Seaborn
- Plotly

---

## Dataset

- Titanic Dataset (CSV)

---

## Steps Performed

### 1. Imported Required Libraries
Imported Pandas, Matplotlib, Seaborn, and Plotly for data analysis and visualization.

### 2. Loaded the Dataset
Loaded the Titanic dataset into a Pandas DataFrame.

### 3. Explored the Dataset
- Displayed the first five rows.
- Checked dataset information.
- Viewed data types.
- Identified missing values.

### 4. Generated Summary Statistics
Calculated:
- Mean
- Median (50%)
- Standard Deviation
- Minimum
- Maximum
- Quartiles

using the `describe()` function.

### 5. Created Histograms
Visualized the distribution of:
- Age
- Fare

### 6. Created Boxplots
Detected outliers in:
- Age
- Fare

### 7. Generated Pairplot
Compared relationships among selected numerical features:
- Survived
- Pclass
- Age
- Fare

### 8. Generated Correlation Matrix
Created a heatmap to visualize correlations between numerical features.

### 9. Made Feature-Level Observations
Observed:
- Most passengers were between 20 and 40 years old.
- Fare distribution is right-skewed.
- Fare contains several outliers.
- Passenger class influences fare.
- Age contains missing values.
